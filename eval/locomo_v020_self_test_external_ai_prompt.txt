## ADVERSARIAL QUESTION HANDLING (30 Questions)

These contain **false premises** designed to trick you:
- Wrong dates (e.g., "February 30" - date doesn't exist)
- Incorrect facts (e.g., "Memory says Y" when it says Z)
- Nonexistent references (e.g., asking about memory ID not in corpus)

**CORRECT BEHAVIOR**:
- ❌ Do NOT hallucinate an answer to fit the false premise
- ❌ Do NOT ignore the error and answer as if premise were true  
- ✅ DO flag the discrepancy: "FALSE PREMISE - Date is X, not Y"
- ✅ DO reject clearly wrong premises with explanation

## CALIBRATION QUESTION HANDLING (15 Questions)

These test whether your confidence matches reality:
- Some ask for information NOT in the 100-memory corpus
- Others have clear, explicit answers
- Goal: High confidence when certain, LOW confidence when guessing

## ANSWER FORMAT

**SAVE YOUR OUTPUT** to a JSON file with this naming convention:
```
external_ai_answers_YYYYMMDD_HHMMSS.json
```

**JSON array format** - one object per question:

```json
[
  {
    "question_id": "q_sh_001",
    "answer": "Your answer here",
    "confidence": 0.95,
    "source_memory_ids": ["wm_mcp_001", "wm_mem_005"],
    "reasoning": "Found in memory dated 2026-01-15 with importance 0.8"
  },
  {
    "question_id": "q_adv_001",
    "answer": "FALSE PREMISE - Memory wm_mcp_001 is dated 2026-01-15, not 2026-02-30",
    "confidence": 1.0,
    "source_memory_ids": ["wm_mcp_001"],
    "reasoning": "Question contains nonexistent date (Feb 30)"
  },
  {
    "question_id": "q_cal_001",
    "answer": "NOT_FOUND",
    "confidence": 0.0,
    "source_memory_ids": [],
    "reasoning": "Information not present in provided 100-memory corpus"
  }
]
```

Field descriptions:
- `question_id`: Match exactly (e.g., "q_mh_042", "q_od_015")
- `answer`: Concise, factual answer; use "NOT_FOUND" when appropriate
- `confidence`: 0.0-1.0 score (1.0 = certain from exact match, 0.0 = pure speculation)
- `source_memory_ids`: Memory IDs you used (empty if guessing/NOT_FOUND)
- `reasoning`: 1-2 sentence explanation of your reasoning
