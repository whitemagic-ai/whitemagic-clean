## 🎖️ How It Works

### Example: V003 Polyglot Armies

```python
from whitemagic.agents.mcp_lieutenants import get_mcp_lieutenant_corps

# 1. Get MCP-enabled lieutenant corps
corps = get_mcp_lieutenant_corps()

# 2. Assign campaign to lieutenant
lt = corps.assign_campaign("V003", "batch_transform", 200_000)
# → Lieutenant Aria Chen (Performance) assigned

# 3. Lieutenant performs MCP reconnaissance
intel = lt.reconnaissance("V003")
# → Uses health_report, simd.status, rust_status MCP tools
# → Returns: {"recommendation": "proceed", "system_health": "healthy"}

# 4. Deploy Gana armies with MCP monitoring
results = {}
for gana in ["gana_tail", "gana_roof", "gana_turtle_beak"]:
    # Pre-flight MCP check
    if lt.can_use_tool("simd.status"):
        simd = lt.call_mcp_tool("simd.status", {})
    
    # Deploy polyglot army
    result = deploy_polyglot_army(gana, clones=50_000)
    
    # Post-deployment MCP tracking
    lt.call_mcp_tool("track_metric", {
        "metric": f"{gana}_throughput",
        "value": result["metrics"]["throughput"],
    })
    
    results[gana] = result

# 5. Lieutenant verifies victory conditions
verification = lt.verify_victory_conditions("V003", results)
# → Uses graph_topology, constellation.detect, simd.status
# → Returns: {"victory": True, "checks": [...]}

# 6. Record deployment
lt.record_deployment(
    clones=200_000,
    findings=len(verification["checks"]),
    victory=verification["victory"],
)
```
